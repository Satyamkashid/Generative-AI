# GenAI Learnings

This repository demonstrates Generative AI workflows using LangChain, Streamlit, and integrations with OpenAI, Anthropic, HuggingFace, and Google GenAI APIs. It contains example document loaders, model adapters, and chain patterns to help prototype and learn common patterns.

## Quick start (Windows)

1. Clone and open repository:
   ```powershell
   git clone <your-repo-url>
   cd "d:\GENAI_LEARNING\Gen-Ai\Generative-AI"

2. Create and activate a virtual environment:
   windows:
      python -m venv .venv
      .venv\Scripts\activate

3. Install dependencies:
      pip install -r requirements.txt

4. Configure environment variables:
      Copy .env.example to .env and fill required keys:
